## Install Zoom Desktop Client

1. Open your browser and go to zoom.us/download.
1. Locate the Zoom Desktop Client section and click Download.
    > **Note:** This saves the installer file to your device.  
2. Run the Installer:
    * On Windows: Double-click `ZoomInstaller.exe`.
    * On macOS: Double-click `Zoom.pkg`.
3. Follow the Setup Wizard.
4. Accept the license terms.
5. Choose your installation directory to add the program files.
6. Click **Install** and wait for setup to complete.
7. Click **Finish** or **Close** once the installation is complete.
8. Open Zoom from your desktop or Applications folder.
```
npm install
```
